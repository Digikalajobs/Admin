# Digikala Landing Upgrade

نسخه آپگرید شده سایت تک‌صفحه‌ای:

- اسلایدهای شناور بدون اسکرول اضافی
- افکت ذرات پس‌زمینه
- اعتبارسنجی شماره موبایل ایران
- نمایش دکمه دانلود بعد از ورود شماره
- ارسال اختیاری شماره به تلگرام با Netlify Functions

## راه‌اندازی

1. فایل‌ها را روی GitHub آپلود کن.
2. پروژه را در Netlify Deploy کن.
3. اگر می‌خواهی شماره‌ها به تلگرام ارسال شوند، در Netlify این متغیرها را بساز:

TELEGRAM_BOT_TOKEN
TELEGRAM_CHAT_ID

4. بعد از اضافه‌کردن متغیرها، Redeploy بزن.

## لینک دانلود APK

دکمه دانلود به این مسیر وصل است:

downloads/app.apk

فایل APK واقعی را با همین اسم داخل پوشه downloads قرار بده.

اگر فعلاً APK نداری، فایل PUT_APK_HERE.txt فقط راهنماست.
